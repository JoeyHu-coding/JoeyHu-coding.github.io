# Image Placement Guide

Use these folders for page images:

- `assets/profile/` - portrait, profile photo, homepage identity image
- `assets/research/` - concept diagrams, research maps, framework images
- `assets/publications/` - paper figures, graphical abstracts, report covers
- `assets/projects/` - funded research, sponsored work, enterprise or engineering project images
- `assets/portfolio/` - design portfolio drawings, renderings, maps, boards
- `assets/archive/` - sketches, field notes, talks, workshops, older material

Recommended image behavior:

- Keep homepage images small and personal; use one profile image first.
- Use captions for research, publication, project, and portfolio images.
- Prefer original images, self-made diagrams, or properly licensed images.
- Keep filenames simple, for example `profile.jpg`, `research-framework-01.jpg`, or `project-title-map.jpg`.
- Replace placeholder blocks in the HTML with `<img src="..." alt="...">` when final images are ready.
